To run the following programs set
'Initial Orders 1' on the Edsac menu

   Squares
   Primes

To run the following programs set
'Initial Orders 2' on the Edsac menu

   Cubes
   Hello
   Oxo
   TPK

The behaviours of Squares, Primes, Cubes,
Hello, and Oxo are self-explanatory.

See Edsac Documentation for a 
description of the TPK program.

There are more programs in the Contributed 
Programs folder.
